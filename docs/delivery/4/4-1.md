## Requirements

1. Go models in `api/internal/models/progress.go`:
   - `LessonProgress` — single lesson's progress (lesson_id, status, started_at, completed_at, notes)
   - `TopicProgress` — response for `GET /api/progress/topics/:id` (topic_id, lessons []LessonProgress)
   - `ProgressSummary` — response for `GET /api/progress/summary` (total_lessons, completed_lessons, completion_percentage, active_topics)
   - `UpdateProgressInput` — request body for `PUT /api/progress/lessons/:id` (status, notes)
   - Status values: `not_started`, `in_progress`, `completed` (matching DB CHECK constraint)

2. TypeScript types in `web/src/api/types.ts`:
   - `LessonProgress` — mirrors Go model
   - `TopicProgress` — mirrors Go model
   - `ProgressSummary` — mirrors Go model
   - `UpdateProgressInput` — request body type
