## Requirements

1. Create `web/` directory at project root with React + TypeScript + Vite scaffold
2. Install dependencies: `react`, `react-dom`, `typescript`, `@types/react`, `@types/react-dom`
3. Configure Tailwind CSS v4 with sensible defaults
4. Configure Vite dev server proxy: `/api` → `http://localhost:8080`
5. Add `pnpm` workspace or standalone `package.json` in `web/`
6. App renders a basic page confirming setup works
7. `pnpm dev` starts the dev server; `pnpm build` produces production bundle
8. Add `.gitignore` entries for `node_modules/`, `dist/`

## Implementation Plan

1. Create `web/` directory
2. Initialise with `pnpm create vite . --template react-ts`
3. Install Tailwind CSS v4: `pnpm add tailwindcss @tailwindcss/vite`
4. Configure `vite.config.ts` with Tailwind plugin and API proxy
5. Update `web/src/index.css` to import Tailwind
6. Create minimal `App.tsx` with a Tailwind-styled heading
7. Verify `pnpm dev` serves the app and proxy forwards `/api/health` to Go backend
8. Verify `pnpm build` succeeds
